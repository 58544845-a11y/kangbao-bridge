# Kangbao Bridge - 桥牌开叫互动测试

这是一个用于学习和练习桥牌开叫（CCBA 自然体系）的互动测试系统，包含初级和中级题库，用户通过输入开叫来进行答题，系统会给出即时反馈与解释。

功能:
- 随机生成 13 张手牌
- 计算高花点(HCP)与分配点
- 支持 CCBA 自然体系的常见开叫（初级与中级）
- 输入开叫并获取即时判定与规则说明
- 成绩统计与进度展示

如何运行:
1. 克隆仓库或下载文件
2. 打开 `index.html`（支持 GitHub Pages）
3. 选择难度并开始练习

文件说明:
- `index.html` - 主界面
- `styles/main.css` - 样式
- `scripts/quiz.js` - 测试交互逻辑（已创建）
- `scripts/evaluation.js` - 手牌评估
- `scripts/bidding.js` - CCBA 开叫规则判断
